<실행>

1. 고객
-> 첫 화면의 START 버튼을 눌러 시작
-> 회원가입을 위한 "Join" 클릭
-> 이름, 비밀번호, 전화번호를 입력 후 '중복확인' 클릭
-> 중복확인에 성공을 해야만 회원가입 완료 가능, 빈칸이 존재하면 회원가입 실패
-> 가입한 회원 정보를 입력 후 "Login" 클릭
-> 시스템에 이미 저장되어있는 음식점 리스트가 화면에 출력됨
-> 예약을 원하는 음식점 선택
-> 각 메뉴 옆에있는 콤보박스로 음식 갯수 조정 (수량이 모두 0일 시 오류창)
-> '예상조리시간' 클릭 시 예상조리시간 확인 가능
-> '예약' 버튼 클릭
-> 지금 당장 가길 원하면 'NOW' 클릭 -> 예약 완료
-> 나중에 가길 원하면 'LATER' 클릭 -> 날짜/시간 선택 후 '예약' 클릭 (과거시간 선택시 오류창) -> 예약 완료
-> '예약확인' 클릭
-> 취소할 예약이 있으면 해당 예약 옆 '취소' 버튼 클릭 (NOW 예약은 취소시 오류창)
-> 취소하시겠습니까? 에 "예" 버튼 클릭 시 예약이 삭제됨
-> '이전' 버튼으로 돌아옴

2. 매니저
-> 첫 화면의 START 버튼을 눌러 시작
-> 매니저 로그인 정보를 입력한 후 'Login' 클릭
-> 삭제할 음식점 버튼을 눌러 음식점 삭제 가능
-> 오른쪽 아래 '추가' 버튼 클릭
-> 추가할 음식점 이름 작성
-> '추가' 클릭 (빈칸이면 오류창)
-> 추가 완료
